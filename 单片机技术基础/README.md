# 单片机技术基础

本目录按课程资料原始用途整理为三部分，避免把同一个实验的 Keil、Proteus 和 Multisim 文件拆散。

## 目录结构

- `单片机平时作业/`：课程平时作业、实验报告格式、实验讲义及已获同意公开的样本工程。
- `单片机实习项目/`：单片机实习相关项目，按实验项目组织。
- `单片机复习资料/`：期末复习题库。

## 实习项目对应关系

- `实验一最小系统板/`：8052 Multisim 仿真文件。
- `实验二秒表设计/`：Keil C51 源码、工程文件和 `.hex`。
- `实验三交通灯控制器设计/`：Keil C51 源码、工程文件、`.hex` 和现象截图。
- `实验四简易数字电压表设计/`：Keil C51 源码、工程文件和 `.hex`。
- `实验五基于TLC5615信号发生器设计/`：Keil C51 源码、工程文件和 `.hex`。
- `实验六多功能LCD时钟/`：Keil、Proteus、Multisim 工程及参考资料。
- `案例/`、`截图与实物图/`、`实习资料/`、`实习报告/`：实习项目配套材料。

## 软件版本

- Keil 工程使用较旧的 C51/Keil 环境保存，建议使用 Keil C51 7.50a 或兼容版本。
- `单片机平时作业/` 中的 Proteus 项目来自课程样本，建议使用 Proteus 6.9 或兼容版本打开。
- `单片机实习项目/` 中的 Proteus 项目使用 Proteus 8.17 完成，建议使用 Proteus 8.17 或兼容版本打开。
- Multisim 工程文件使用 Multisim 14 保存。

## 说明

`.hex` 文件是编译结果，用于 Proteus 关联和烧录复现参考；源码与工程文件仍是主要内容。讲义已按用途拆分：`单片机平时作业/单片机实验讲义.pdf` 对应平时实验，`单片机实习项目/实习资料/单片机实习讲义.pdf` 对应集中实习。平时作业样本已取得公开授权。已排除软件安装包、便携 Git/GitHub CLI、缓存、备份、工作区文件、Keil 编译中间产物、无扩展名输出文件、原始视频和含本机路径的软件界面截图。

`单片机实习项目/案例/` 下的样本 PDF 已做脱敏处理：替换封面页并清理 PDF 文档元数据；未脱敏原始文件不直接公开。
