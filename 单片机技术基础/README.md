# 单片机技术基础

本目录收录单片机技术基础课程的平时作业、复习题库和实习工程文件。

## 内容

- `平时作业/`：课程平时作业文档。
- `复习资料/`：期末复习题库。
- `实习资料/`：实习安排和报告模板。
- `Keil工程/`：实习相关 C51 源码、Keil 工程文件和少量 `.hex` 编译结果。
- `Proteus工程/`：实验六 Proteus 仿真工程。
- `Multisim工程/`：8052 与实验六 Multisim 仿真文件。

## 软件版本

- Keil 工程使用较旧的 C51/Keil 环境保存，建议使用 Keil C51 7.50a 或兼容版本。
- Proteus 工程建议使用 Proteus 6.9 或兼容版本打开。
- Multisim 工程文件使用 Multisim 14 保存。

## 说明

`.hex` 文件是编译结果，仅供复现和烧录参考；源码与工程文件仍是主要内容。软件安装包、便携 Git/GitHub CLI、同学样本、个人报告、视频、缓存和编译中间产物均未纳入仓库。
