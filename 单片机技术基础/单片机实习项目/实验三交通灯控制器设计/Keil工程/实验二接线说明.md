# 实验二接线说明

## 按讲义附录接线

这版代码不使用 P2 口，避免和你已经按附录焊接/插线的电路冲突。

## 三色灯输出

沿用原实验代码里的端口关系：

- 绿灯接 P1.2，对应 `0x04`。
- 黄灯接 P1.3，对应 `0x08`。
- 红灯接 P1.4，对应 `0x10`。

代码里使用 `sbit` 单独控制这三个引脚，不再整口写 `P1 = 0x04 / 0x08 / 0x10`，这样不会影响拨码开关输入。

## 拨码开关输入

为了避开三色灯占用的 P1.2、P1.3、P1.4，拨码开关使用：

- SW2-1 / S0 接 P1.5：强制切到绿灯。
- SW2-2 / S1 接 P1.6：强制切到黄灯。
- SW2-3 / S2 接 P1.7：强制切到红灯。

拨码开关按低电平有效处理：

- 高电平 / 1 / Debug 里打勾：没有触发。
- 低电平 / 0 / Debug 里不打勾：该开关有效。

## 数码管显示模块

保持实验一的显示模块接线：

- P3.0 接 DATA。
- P3.1 接 CLK。

## 定时说明

- 使用定时器0，方式1，16 位定时器。
- 12MHz 晶振下，定时器0每 10ms 中断一次。
- 累计 100 次 10ms 中断，得到约 1 秒。
- 倒计时每秒更新一次。

关键代码：

```c
void timer0Isr(void) interrupt 1
```

这是定时器0中断服务函数。
