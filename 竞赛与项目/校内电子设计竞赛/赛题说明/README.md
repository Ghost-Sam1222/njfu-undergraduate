# 赛题说明

## 比赛背景

南京林业大学 2026 年校内电子设计竞赛。题目本质是**巡线小车**的进阶版本——在黑胶带铺成的环形赛道上，小车需自主完成特定圈数的行驶并准确停止。

## 第一题：绕行一圈

**要求**：小车从起点出发，沿黑线赛道行驶一圈后自动停止。

**解题思路**：
- 8 路红外传感器检测黑线位置
- 基础巡线逻辑（`xunji.c` 中的 `track1()`）
- 检测到所有传感器同时压线（回到起点）时刹车

## 第二题：绕行 N 圈并停止

**要求**：小车沿赛道行驶指定圈数（如 4 圈），途中需识别节点和标志，最后准确停在终点。

**解题思路**：
- 利用红外传感器检测「白节点」（所有传感器同时看到白色，即赛道交叉处）
- 用计数器 + 防抖逻辑累计经过的节点数
- 配合 IMU 惯性测量单元获取偏航角，实现精确转弯
- PID 闭环控制车速和转向角度
- 到达目标圈数后停车

## 场地示意

```
       ┌──────────────────┐
       │   黑胶带赛道      │
       │  ┌────────────┐  │
       │  │  (环形)     │  │
       │  │   ← 小车   │  │
       │  │   巡线行驶  │  │
       │  └────────────┘  │
       │  起点/终点 ★     │
       └──────────────────┘
```

## 附加题：K230 视觉识别

利用 K230 摄像头模块完成：
- **颜色识别**：红 / 绿 / 蓝
- **形状识别**：圆形 / 矩形 / 三角形（含等边三角形判定）
- **数码管识别**：识别 7 段数码管显示的数字（0-9）
