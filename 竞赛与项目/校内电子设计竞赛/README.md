# 校内电子设计竞赛 — 基于 TI MSPM0 的智能小车

## 一句话简介

用 **TI MSPM0G3507** 单片机做主控，搭载 **K230 摄像头**做图像识别，制作一台能**巡线行驶、多圈计数、自主停止**的智能小车。最终完成两轮比赛任务。

## 你能从本项目学到什么

| 模块 | 涉及知识点 |
|------|-----------|
| 固件源码/user/ | MSPM0 外设驱动、主循环架构、中断编程 |
| 固件源码/ml_libs/ | GPIO/PWM/UART/I2C/TIM 逐层封装思想 |
| 固件源码/code/ 和 user/_pid.c | PID 控制算法（增量式/位置式/串级） |
| 图像识别/ | K230 视觉模块：颜色识别、形状检测、数码管识别 |
| 硬件方案/ | 芯片选型思路、传感器接线、电机驱动 |

## 项目结构

```
├── README.md           ← 你在这里
├── 赛题说明/            ← 比赛考什么
├── 硬件方案/            ← 用了什么硬件，怎么接线
├── 设计报告/            ← 完整设计报告 PDF
├── 固件源码/            ← MSPM0 单片机代码
│   ├── user/           ← 主控逻辑（巡线、PID、传感器）
│   ├── ml_libs/        ← 硬件驱动库（GPIO/PWM/UART/I2C 等）
│   └── code/           ← PID 算法模块（早期版本）
├── 图像识别/            ← K230 摄像头 Python 脚本
└── 作品展示/            ← 现场照片和演示视频
```

## 开发环境

- **IDE**：Keil MDK（ARM Compiler 6）
- **MCU**：TI MSPM0G3507 @ 80MHz
- **SDK**：TI MSPM0 SDK v1.20+（[TI 官网下载](https://www.ti.com/tool/MSPM0-SDK)）
- **视觉模块**：K230（CanMV / MicroPython）
