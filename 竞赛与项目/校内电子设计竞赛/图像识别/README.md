# 图像识别 — K230 视觉模块

三组 Python 脚本运行在 **K230（CanMV / MicroPython）** 摄像头模块上，通过 UART 将识别结果传回 MSPM0。

## 文件说明

| 文件 | 功能 | 算法要点 |
|------|------|----------|
| `color_shape.py` | **颜色 + 形状识别** | LAB 颜色阈值 + OpenMV `find_circles()`/`find_rects()` |
| `find_triangles.py` | **三角形检测**（含等边判定） | 轮廓检测 + blob 检测融合，边长和内角计算 |
| `digit_7seg.py` | **7 段数码管识别**（0-9） | 7 段 ROI 映射 + 亮度比值阈值（支持红/绿/蓝/暗四种颜色模式） |

## 运行环境

- **硬件**：K230 开发板 + 摄像头模组
- **固件**：CanMV（MicroPython 兼容）
- **与 MSPM0 通信**：UART 115200bps，帧格式 `[0x75][addr][dataL][dataH][0x04]`
