# 硬件方案

## 主控芯片

**TI MSPM0G3507**（Arm Cortex-M0+ @ 80MHz）

- 128KB Flash / 32KB SRAM
- 丰富的定时器和通信外设
- 低成本、低功耗，适合嵌入式竞赛

## 硬件清单（BOM）

| 模块 | 型号/参数 | 用途 |
|------|-----------|------|
| 主控板 | MSPM0G3507 开发板 | 整车控制核心 |
| 电机驱动 | TB6612 或 L298N | 驱动两个直流电机 |
| 直流电机 ×2 | 带编码器（如 JGA25-370） | 车轮驱动 |
| 红外传感器 | 8 路 TCRT5000 阵列 | 黑线检测 |
| IMU 模块 | 串口六轴陀螺仪 | 偏航角测量 |
| OLED 显示屏 | 0.96" 128×64 SSD1306 | 调试信息显示 |
| 超声波模块 | HC-SR04 | 障碍物测距（预留） |
| 摄像头 | K230（CanMV） | 颜色/形状/数码管识别 |
| 电源 | 7.4V 2S 锂电池 | 整车供电 |

## 传感器接线参考

> 具体引脚定义见 `固件源码/user/main.c` 初始化代码和 `固件源码/ml_libs/` 各驱动头文件。

| 外设 | 接口 | 关键引脚 |
|------|------|----------|
| 左电机 PWM | TIMG6_CH0/CH1 | PB6/PB7 |
| 右电机 PWM | TIMG6_CH2/CH3 | 见 ml_motor.h |
| 左编码器 | EXTI | PA12/PA13 |
| 右编码器 | EXTI | PA24/PA15 |
| 红外传感器 ×8 | GPIO 输入 | D1-D8（见 xunji.h） |
| IMU | UART1（9600bps） | 见 IMU.h |
| OLED | 软件 I2C | PB2/PB3 |
| 超声波 | GPIO | TRIG=PB9, ECHO=PA10 |
| K230 通信 | UART0（115200bps） | 见 serical.c |
| 蜂鸣器 | GPIO | PB8 |
| 按键 ×2 | GPIO | PA17（模式）, PA27（设置） |
