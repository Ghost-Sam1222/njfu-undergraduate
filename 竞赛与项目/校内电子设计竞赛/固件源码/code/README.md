# code/ — PID 算法模块（早期版本）

⚠️ **注意**：本目录的 `pid.c` 是**项目早期的 PID 实现**，最终版本在 `../user/_pid.c`。

## 保留原因

两版 PID 并置，可以对比学习「速度 PID」和「位置角度 PID」的演进过程：

| | code/pid.c（早期） | user/_pid.c（最终） |
|---|---|---|
| 控制对象 | 电机速度 | 电机速度 + 角度 |
| PID 类型 | 增量式速度 PID | 增量式速度 + 位置式角度 + 串级 |
| 低通滤波 | 有（α=0.3） | 无 |
| 积分限幅 | 有 | 无 |
| 用途 | 基础巡线 | 带 IMU 的高级导航 |

## 文件说明

| 文件 | 功能 |
|------|------|
| `pid.c` | 速度 PID（增量式）+ 位置角度 PID + 巡线 PD |
| `pid.h` | 函数声明 |

## 学习建议

1. 先看 `user/_pid.c` 理解最终方案
2. 再回来看 `code/pid.c` 理解早期方案
3. 对比两者，思考为什么从「速度 PID」演进到「速度+角度串级 PID」
