# user/ — 主控应用层

本目录是**整车控制的核心代码**，包括入口、巡线、PID、传感器驱动业务逻辑。

## 文件说明

### 入口与调度

| 文件 | 功能 | 适合谁看 |
|------|------|----------|
| `main.c` | 系统初始化 + 主循环 + 10ms 定时中断调度 | **入门必读**：理解 MSPM0 项目架构 |
| `isr.c` | 中断服务函数：定时器、串口、蜂鸣器 | 理解中断注册方式 |

### 巡线导航

| 文件 | 功能 | 核心看点 |
|------|------|----------|
| `xunji.c` / `xunji.h` | **巡线逻辑核心**：4 种模式 track1~track4 | 8 路红外读取、节点检测与防抖、IMU 辅助转弯 |
| `track.c` / `track.h` | 辅助巡线函数（已基本被 xunji.c 替代） | 早期版本代码，可对比学习 |

### 控制算法

| 文件 | 功能 | 核心看点 |
|------|------|----------|
| `_pid.c` / `_pid.h` | **PID 控制器**：增量式速度 PID + 位置式角度 PID + 串级控制 | 增量式 vs 位置式 PID 的区别、角度归一化 |

### 传感器驱动

| 文件 | 功能 | 核心看点 |
|------|------|----------|
| `encoder.c` / `encoder.h` | **编码器**：两路正交编码器读取 + 方向判断 | 外部中断输入捕获、编码器方向解码 |
| `IMU.c` / `IMU.h` | **串口 IMU**：UART1 接收，解析 Z 轴偏航角 | 串口协议解析（0x55 + ID 0x53） |
| `hcsr04.c` / `hcsr04.h` | **超声波**：HC-SR04 测距（预留功能） | 触发脉冲 + 回波测时 |
| `serical.c` / `seriacl.h` | **K230 通信**：UART0 收发图像识别结果 | 帧协议 `[0x75][addr][data][0x04]`，注意头文件拼写错误 |

### 其他

| 文件 | 功能 |
|------|------|
| `key.c` / `key.h` | 按键消抖：PA17 模式切换 / PA27 设置切换 |
| `buzzer_light.c` / `buzzer_light.h` | 蜂鸣器 + LED 指示，500ms 自动关闭 |
| `ti_msp_dl_config.c` / `.h` | **SysConfig 生成的管脚/时钟配置**（编译必需） |
| `startup_mspm0g350x_uvision.s` | 启动文件 + 向量表（TI 标准文件） |
| `mspm0g3507.sct` | 链接脚本：FLASH/SRAM 地址分配 |
| `project.uvprojx` | **Keil MDK 工程文件**（双击打开项目） |
| `empty.syscfg` | SysConfig 配置文件 |

## 学习路径建议

```
第 1 步：main.c → 看系统怎么初始化、主循环怎么跑
第 2 步：xunji.c → 看巡线逻辑怎么实现
第 3 步：_pid.c → 看 PID 怎么控制速度和角度
第 4 步：encoder.c → 看编码器怎么读取速度
第 5 步：serical.c → 看怎么和 K230 通信
```
