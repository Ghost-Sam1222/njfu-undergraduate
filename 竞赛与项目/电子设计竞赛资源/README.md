# 电子设计竞赛资源

本目录汇总了参加全国/江苏省大学生电子设计竞赛（TI 杯）所需的各种资源——从官方通知到硬件驱动，从历年赛题到培训讲义。

## 目录

| 目录 | 内容 | 适合谁 |
|------|------|--------|
| `官方通知/` | 2024-2026 年竞赛通知、测试安排、测评手册 | **新手必看**：了解竞赛时间线和流程 |
| `历年赛题/` | 2024 江苏省赛 + 2025 全国赛题目 | 练手、了解出题风格 |
| `报告模板/` | 2021/2024/2025 国赛/省赛报告格式 | 写报告时参考 |
| `培训资料/` | 赛前培训讲义、电机驱动对比、FPGA 入门 | **核心**：备赛知识储备 |
| `校内训练赛/` | 2026 年校内训练赛题 + 报告格式 + 生成脚本 | 校内选拔参考 |

## 培训资料详解

### 赛前培训
- 2025/2026 年信息与通信类赛题知识储备 PPT
- 运放与模拟滤波器设计（单电源/双电源/有源滤波器）
- STM32F407 数据手册（中英文）

### 硬件驱动及使用说明
教师提供的 **25 个 MSPM0G3507 例程**，涵盖：
串口通信、PWM、ADC、定时器、编码器、IMU、GPS、气压计、超声波、光流、蓝牙、灰度传感器、PID 等。是 TI MSPM0 平台最实用的代码参考。

此外还有 MSP430F5529 相关资料和电机驱动模块手册。

### 教程链接
TI 杯竞赛准备指南 + 大连理工大学李胜铭老师视频教程入口。

### 知识问答
2025 年竞赛题目答疑记录（4 份），了解组委会对赛题的官方解读。

## 校内训练赛

2026 年 5 月 17 日举办的校内选拔赛资料：
- `赛题与格式/`：竞赛题目 + 报告书写格式
- `build_report_docx.py`：Python 脚本，自动生成竞赛报告文档
- `lastone.py`：报告辅助生成脚本
