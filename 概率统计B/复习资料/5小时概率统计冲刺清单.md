# 5小时概率统计冲刺清单

## 先抓大头

按当前真题重复率，优先级如下：

1. 一维连续型随机变量：密度归一化、分布函数、概率、变量函数密度。
2. 二维随机变量：联合分布/联合密度、边缘分布、独立性、区域积分或联合表。
3. 参数估计：矩估计、最大似然估计。
4. 正态总体统计推断：置信区间、假设检验、抽样分布。
5. 基础概率公式：全概率、贝叶斯、事件运算、二项/泊松/均匀/正态、切比雪夫。
6. 数字特征：期望、方差、协方差、相关系数。
7. 回归只背选择题公式，不深练。

## 5小时安排

### 0:00-0:20 公式急救

- 全概率：P(B)=sum P(A_i)P(B|A_i)
- 贝叶斯：P(A_i|B)=P(A_i)P(B|A_i)/P(B)
- 切比雪夫：P(|X-EX|>=e)<=DX/e^2；P(|X-EX|<e)>=1-DX/e^2
- D(aX+bY+c)=a^2DX+b^2DY+2abCov(X,Y)
- Cov(X,Y)=rho sqrt(DX) sqrt(DY)
- 独立时：联合=边缘乘积，Cov=0，D(X+Y)=DX+DY
- 正态均值区间：方差已知用 u，方差未知用 t(n-1)
- 回归：直线过样本均值点，a=y_bar-b x_bar

模板：

- 区域概率：画区域，定积分上下限。

## 可以战略放弃或低投入

- 回归：只记 a=y_bar-b x_bar，直线过 (x_bar,y_bar)。
- 复杂保险/赔付应用题：若时间不够，只做第一问概率/期望。
- 很绕的抽样分布选择题：记住正态总体的常见结论即可。

## 考场顺序

1. 先做选择填空里一眼能套公式的。
2. 大题先做一维连续、二维、参数估计。
3. 推断题按模板写满步骤，哪怕数值没算完也拿过程分。
4. 最后回头处理概率应用题和长叙事题。
